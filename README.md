# noir

*A silent obsidian study chamber.*

noir is a cinematic dark-academia productivity web app built for deep focus, quiet studying, and immersive rituals.
Instead of bright dashboards and gamified productivity systems, noir embraces minimalism, emptiness, typography, and atmosphere.

The experience is designed to feel less like software and more like a private midnight archive.

---

# Features

## Hourglass

A cinematic Pomodoro timer designed for immersive focus sessions.

* Full-screen timer experience
* Editable `HH:MM:SS` duration
* Persistent countdown state
* Pause / resume / reset
* Session tracking
* Minimal distraction-free interface

---

## Memento

A minimalist stopwatch experience.

* Full-screen stopwatch
* Accurate elapsed time tracking
* Persistent state
* Pause / resume / reset
* Optional lap support

---

## Chronicles

A quiet space for notes and reflections.

* Minimal writing environment
* Autosave support
* Persistent drafts
* Distraction-free manuscript aesthetic

---

## Rites

A restrained task system for study intentions.

* Minimal checkbox tasks
* Persistent task storage
* Lightweight interaction design
* Clean monochrome interface

---

## Dossier

A statistics archive for tracking progress.

* Focus session history
* Total study hours
* Streak tracking
* Weekly and monthly analytics
* Minimal monochrome data visualizations

---

## Archive

A collection of unlockable environments and relics.

### Themes / Rooms

Unlock atmospheric study spaces such as:

* Rainroom
* Midnight Archive
* The Observatory
* Blackwood Dormitory

### Collectibles

Unlock monochrome relics through consistent studying.

Examples include:

* hourglasses
* manuscripts
* fountain pens
* candles
* observatory charts
* ravens
* antique keys

# Aesthetic Philosophy

noir is intentionally:

* monochrome
* cinematic
* typography-focused
* minimal
* quiet
* immersive

The interface avoids:

* flashy UI
* productivity startup aesthetics
* bright colors
* gamification overload
* cluttered dashboards

The visual direction is inspired by:

* obsidian stone
* cathedral silence
* old manuscripts
* midnight libraries
* rain-soaked study rooms
* dark academia
